# Gabits — Goal & Habit Tracking App

## Overview
Dark-mode React Native (Expo) app for tracking goals broken into micro-goals, with circular progress visualization.

## Stack
- Expo SDK 54, expo-router file-based navigation
- TypeScript strict
- Axios for API (transformRequest stringifies JSON)
- expo-secure-store (mobile) + localStorage (web) for token persistence
- react-native-svg for animated circular progress
- expo-linear-gradient, expo-haptics

## Backend
- Base URL via `EXPO_PUBLIC_API_URL`
- Auth: POST /api/signup, POST /api/auth/login, GET /api/auth/me
- Goals: CRUD at /api/goals
- Categories: /api/categories
- Micro-goals: nested create, standalone toggle/update/delete

## Navigation Structure
- `app/_layout.tsx` — root with AuthProvider + ErrorBoundary + Stack (headerShown false)
- `app/index.tsx` — Redirect based on auth state
- `app/auth/{_layout,login,signup}.tsx` — redirects to /tabs if authed
- `app/tabs/{_layout,index,categories}.tsx` — bottom tabs: Dashboard + Categories
- `app/goal/{_layout,[id],create}.tsx` — goal detail + create/edit (edit uses ?id= query)
- `app/+not-found.tsx`

## Design Tokens (`src/theme.ts`)
- Dark palette: bg #0A0A0A, surface #1A1A1A
- Primary #7C3AED, accent #EC4899 (gradient)
- Priority colors: foundational=#10B981, priority=#F97316, important=#6366F1, would-be-nice=#64748B
- Circular progress: SVG with linear gradient, animated stroke-dashoffset (800ms ease-out)

## Key Components
- `CircularProgress` — animated SVG ring with gradient
- `GoalCard` — 2-column grid card with progress ring, category/priority pills, priority stripe
- `GradientButton`, `TextField` (floating label), `ErrorBoundary`

## Key Patterns
- Optimistic micro-goal toggle with rollback on failure
- Pull-to-refresh on Dashboard + Categories
- useFocusEffect to refresh data on screen focus
- Platform-specific confirm dialogs (Alert on mobile, window.confirm on web)
